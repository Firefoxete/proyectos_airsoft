# 🛠️ Airsoft Electronics Hub - Tercio GRT

Bienvenido al repositorio oficial de proyectos electrónicos del equipo **Tercio GRT (Ourense)**. Aquí compartimos nuestros desarrollos diseñados para mejorar el realismo y la jugabilidad en nuestras partidas de Airsoft.

> [!IMPORTANT]
> **Aviso de Propiedad:** Todo el contenido de este repositorio (código, esquemas y diseños) es propiedad exclusiva del equipo **Airsoft Tercio GRT (Ourense)**.

---

## 📂 Estructura de los Proyectos
Cada carpeta de este repositorio es un proyecto independiente que contiene:
* 💻 **Código:** Archivo `.ino` para Arduino.
* 📐 **Esquema:** Diagrama de conexiones y circuito.
* 📸 **Imagen:** Captura visual del prototipo.
* 🛒 **Componentes:** Listado detallado del hardware necesario.
* 🌐 **Simulación:** Enlace interactivo para pruebas online.

---

## 🎮 Catálogo de Proyectos

### 🕒 [Tiempo_Respawn](./Tiempo_Respawn)
Dispositivo para controlar los tiempos de regreso al juego tras ser eliminado.
* **Componentes:** Arduino Nano, LCD 16x2, Pulsador, Buzzer.
* **Simulación:** [🔗 Ver en Tinkercad](https://www.tinkercad.com/things/foAS6nZYLsr-tiempo-respawn/editel?returnTo=%2Fdashboard%2Fdesigns%2Fcircuits&sharecode=CH09XfIobxiP9U-pdws4aHXOTHLoOd0cuFgoujlKaBA)

---

### 💣 [Bomba_cables](./Bomba_cables)
Mecánica de desactivación clásica: identifica y corta los cables correctos bajo presión.
* **Componentes:** Arduino Uno, Cables puente, Resistencias, LED RGB.
* **Simulación:** [🔗 Ver en Tinkercad](https://www.tinkercad.com/things/lNTmmUpoRqc-bombacables/editel?returnTo=https%3A%2F%2Fwww.tinkercad.com%2Fdashboard%2Fdesigns%2Fcircuits&sharecode=928QauZUV2rH9Qx3ao66gCchF22NXgUPbBlpXjFhnr4)

---

### 🏁 [Juego_Dominacion](./Juego_Dominacion)
Sistema de doble cronómetro (tipo ajedrez) para medir el tiempo de control de cada equipo.
* **Componentes:** Arduino Uno, 2x Pulsadores arcade, Pantalla LCD.
* **Simulación:** [🔗 Ver en Tinkercad](https://www.tinkercad.com/things/2HNrbHfdkhT-juego-dominacion/editel?returnTo=%2Fdashboard%2Fdesigns%2Fcircuits&sharecode=-jCWc21kXVyDgKOxSDGB8ut_EpYsWDEZcdbMTDrKG0o)

---

### 🚀 [Lanzadera_misil](./Lanzadera_misil)
Controlador de lanzadera de utilería con secuencias de seguridad y cuenta atrás.
* **Componentes:** Arduino, Interruptor de llave, Teclado matricial 4x4.
* **Simulación:** [🔗 Ver en Tinkercad](https://www.tinkercad.com/things/4ltVQF4jtAc-lanzadera-misil/editel?returnTo=%2Fdashboard%2Fdesigns%2Fcircuits&sharecode=ZSHcWG4IgIgvvCrFEJqQKAfwC8kGl4eL1GU4KtyDoHM)

### ⚡ [Mortero_airsoft](./Mortero_airsoft)
Dispositivo que contabiliza impactos mediante radiofrecuencia para fuego de apoyo.
* **Componentes:** Módulo NRF24L01, Arduino Pro Mini, Acelerómetro.
* **Simulación:** [🔗 Ver en Tinkercad](https://www.tinkercad.com/things/6XrHD7Jd3lc-morteroairsoft/editel?returnTo=%2Fdashboard%2Fdesigns%2Fcircuits&sharecode=5SmwYcFb-S0c8ghVz6g-rIZIfolztk-YUM32ZPl6e98)

### 💥 [Prototipo_granada](./Prototipo_granada)
Granada electrónica para Airsoft con retardo ajustable y aviso sonoro.
* **Componentes:** Batería LiPo 3.7v, ATtiny85 (o Nano), Altavoz piezoeléctrico.
* **Simulación:** [🔗 Ver en Tinkercad](https://www.tinkercad.com/things/iLPm50UtAQW-prototipo-granda/editel?returnTo=%2Fdashboard%2Fdesigns%2Fcircuits&sharecode=dXB6HwwllNdEGLB4LCyAZbt8jnFqSf7HFJsbAUnxxSI)
